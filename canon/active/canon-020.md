## サマリー

### 世界状態の変化

- **デバッグ出力「subsurface_resonance」（初登場・EP-020確定）**: 地下の深い層から来る残留パターンを対象にした解析出力タグ。通常の魔法現象（routing_table・magic_interference_analysis等）とは異なり、精度が低下する。精度低下の理由: 通常は「目の前の魔法現象を視認している状態」でのスキル発動だが、今回は「足元の方向から漏れている感知を追う」という間接的な状況のため（RULE-M08の発動条件を満たしているが精度に影響）

- **「pre-routing_table era」の残留パターン（EP-020確定）**: 現在のマナ経路管理（routing_table、EP-012で確認）が成立するより以前の時代の痕跡が帝都大聖堂地下に存在することが示唆された。バグの「根」が現在見えている表層の破損（routing_table破損等）より深い層にある可能性の布石

- **「READ_ONLY（書き込み権限なし）」の確認（EP-020確定）**: アルの現在の権限では、この深さに対して修正はおろか精度の高い解析も届かない。「elevated authority（上位権限が必要）」（EP-012出力）との接続が示唆される。技術的制約と制度的制約が同じ場所に重なっていることをアルが認識した

- **大聖堂閉架資料の「原因記述の空白」（EP-020確定）**: 大崩壊年代記に原因についての詳細記述が存在しない。「明らかではない」という記述が残っている。これが「記録できなかった」か「記録しなかった」かは不明

### 新規設定

**デバッグ出力フォーマット（EP-020確定）**

```
[SCAN: subsurface_resonance — passive_source]
  depth_estimate: deep (cannot resolve precise layer)
  signal_type: residual_pattern (origin: unknown)
  pattern_age: old (pre-routing_table era — estimated)
  coherence: fragmented
  note: signal does not match current mana_routing schema
        pattern structure unrecognized — older format?
  status: READ_ONLY (no write access at this depth)
```

- 対象: 地下の深い層から来る残留パターン（直接視認ではなく感知方向への間接解析）
- MP消費: 基礎15MP + 追加5MP = 20MP（対象が不明確で深い解析に至れないため追加コストは小）
- 精度低下の理由: RULE-M08「発動条件: 対象の魔法現象を視認している状態」を満たしているが、「足元の地下から漏れてくるもの」という間接的対象のため通常より精度が低下する（RULE-M08の「受動的感知漏れ」とは区別する能動的発動だが、対象の可視性が低い特殊ケース）

**大崩壊年代記に関する確定情報（EP-020）**

- 「修復の記録——ルクス教東部管轄域」: ルクス教視点で書かれた崩壊後の修復記録が閉架書庫に存在する（EP-020で閲覧確認）
- 「修復は地の深いところから行われた」という記述: 公式の教義解釈（女神が世界の根底から恩寵を注ぎ直した）とも読めるが、学術的には「深層と表層の別々の修復」という解釈も存在する（クラウス言及）
- 原因記述の空白: 大崩壊の原因についての詳細記述が年代記に存在しない（「諸説あり」「明らかではない」のみ）

**クラウスの発言（新確定情報）**

- 「学術的には——地の下に残った不整合が別に存在したのではないか、という解釈もある。ただしそれは教義会議の承認を得た解釈ではない」（EP-020確定）
- 「地の下に残った不整合が今も残っている可能性は?」への返答が「興味深い質問だ」のみ（回答回避 or 知らない or 知っていて伏せている——アルには判断不可能）

## 備考

### MP消費の詳細

- EP-019終了時MP: 約97〜104/120
- 4日間の自然回復: +28MP（7MP/日×4日）により上限120/120到達
- EP-020でのデバッグスキル能動的発動: 20MP消費（基礎15MP + 追加5MP）
- EP-020終了時MP: 100/120
- 消耗感の描写: 足の裏から腰にかけての鈍い消耗感あり（INV-A01準拠）

### INV-D01（W-001）対応について

QA W-001で指摘の「間接的感知源への能動的解析」については、以下の通り整理する。

「足元の地下から漏れてくるもの」を対象にした能動的発動は、RULE-M08現行記述の「受動的感知漏れ（大規模魔法異常）」とは区別される特殊ケースである。EP-020実績として「感知方向への間接的能動解析」のケースが初めて確定したが、canon-proposal（p.122-128）が提案するRULE-M08へのサブ項目追記は、今後の追加発動実績を見て editor が評価・判断する。現時点では本備考に記録としてとどめる。

### LOOP-010（W-004）対応について

フィンの「北翼」への外面的反応は、canon-018の「LOOP-010 継続（EP-018）: Last Touched EP-015のまま」という前例と同様に「布石描写として記録するがLast Touchedは変えない」処理を採用した。LOOP-010の本格進展はEP-022予定。

### 「7MP/日」自然回復レート（W-003）について

canon-018備考欄でも採用済みの「7MP/日」という通常消費後の自然回復レートが、本エントリでも前提として機能している（EP-019終了97〜104/120 + 28MP = 125〜132/120 → 上限120/120）。RULE-M01の「一晩の睡眠で完全回復」との整合確認: EP-020は「4日後」という複数日経過のため上限到達は自然。「7MP/日」の仕様明示は今後の判断事項として保留。
