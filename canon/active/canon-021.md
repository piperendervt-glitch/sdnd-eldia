### 世界状態の変化

- **デバッグ出力「deep_trace」（初登場・EP-021確定）**: 目の前の対象（壁の魔法陣）を起点として深層レイヤーを追跡した能動的発動から得られる出力タグ。EP-020の「subsurface_resonance」（地下の受動感知源への間接解析）とは区別する。精度は低い（integrity LOW）。MP消費は深さに応じて追加コスト増

  ```
  [SCAN: deep_trace — subsurface_layer]
    layer: below current mana_routing schema
    coherence: fragmented (partial read)

    partial_data recovered:
      node_id: [REDACTED]
      linked_function: soul_transfer_protocol
      sub_process: outsider_registry
      bridge_component: consciousness_bridge

    flag: data_integrity LOW — names recovered,
          values and context unknown
    note: naming convention differs from auto-generated
          labels — appears intentional
  ```

  | 比較軸 | subsurface_resonance（EP-020） | deep_trace（EP-021） |
  |--------|-------------------------------|----------------------|
  | 起点 | 地下からの受動感知漏れを追う間接解析 | 目の前の対象（壁の魔法陣）を直接視認した上で深層を追跡 |
  | 発動条件 | RULE-M08「視認」の拡張ケース | RULE-M08通常ケース（直接視認）+ 深層追跡 |
  | 精度低下の原因 | 対象の可視性が低い（間接的） | データ自体の劣化（断片的な残留状態） |
  | 追加MP | +5MP | +7MP |

- **意味のある変数名群（EP-021初登場・確定）**: `soul_transfer_protocol`、`outsider_registry`、`consciousness_bridge`。命名規則が自動生成と異なり意図的に見える。LOOP-001（転生の理由）とLOOP-007（デバッグ出力がコードに見える理由）への両側からの布石として機能。「値と文脈は不明」「データ整合性が低い」という制限付き

- **大崩壊以前の施設仮説（EP-021クラウス言及・確定）**: 「外から来る何かを受け入れるための施設だった」という学術的仮説をクラウスが言及。証拠なし・仮説の域を出ないものとして記録

- **魔法陣の歴史（EP-021フィン言及・確定）**: 大崩壊以前の魔法陣は現代より粒度が細かく、記号の意味が半分以上未解読。「最初に整備した集団が消えた」という説と「世界の外から来た集団」という非主流の仮説がある

### deep_trace出力タグについて（QA Warn W-002対応）

`deep_trace`は RULE-M08 の既定規定（発動条件・MP消費レンジ・一対象制約）と矛盾しない「仕様の拡張事例」として扱う。specs/reference/magic_physics.md の RULE-M08 へのサブ項目追記は別途検討する（pending）。

### クラウスの「外から来る何か」発言の設計意図

意図的な情報提供か学術的な共有かをアルが判断できない形で設計（EP-020の「興味深い質問だ」と同様のパターン）。クラウスが何かを知っている可能性と単純に学術的文脈で語っているだけの可能性の両方が読者に開かれたまま維持される。

### LOOP-010（フィンの口ごもり）対応（QA Warn W-001対応）

フィンの「世界の外から来た説に触れて口ごもった」描写はEP-020のEVT-020-11と同様の「外面描写のみ・内面は不明」の処理を採用した。LOOP-010の本格進展はEP-022予定。Last Touched はEP-019のまま維持。open_loops_sync_check.md に理由を明示記録済み（W-001対応）。
