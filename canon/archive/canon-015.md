## サマリー

帝都カエルムの外環に到着した翌日、三人は召喚状を通行証として内環に初入場した。魔法学院でクラウス＝ヴェルナー学院長（NPC-008）と面会し、数週間の短期研修プログラムを受諾した。続いてルクス教大聖堂でベルント司教（NPC-007）と直接面会し、アルが修正経緯を説明した（INV-B03準拠でベルントの真意はアルには伝わらず）。大聖堂の北翼を歩いた際、アルのデバッグスキルが無意識に反応し地下から微弱なノイズを感知した（LOOP-009開始）。帰路に立ち寄った魔法学院図書館でフィンが偶然「異才者の記録」を発見し、アルは数百年前にも異常な理解力で魔法を操る者が複数存在していた可能性を認識した（LOOP-001進展）。

| EVT-015-02 | 三人が召喚状と冒険者カードを提示して内環の関門を通過し、内環に初入場した。滞在期間中有効な通行票三枚を発行された |

| EVT-015-04 | クラウスがフィン＝アルゼスに「叔父君とは別に、自分の目で見なさい」と発言した（クラウスがフィンとベルントの関係を把握していることが示唆された） |
| EVT-015-05 | ルクス教大聖堂でベルント司教（NPC-007）と直接面会した（初対面）。ベルントはアルに修正経緯の説明を求め、アルはラグナ村からリンデンに至る一連の修正の経緯を説明した。ベルントは穏やかに傾聴したが、真の目的（アルの能力の直接評価）はアルには伝わらなかった（INV-B03準拠） |
| EVT-015-06 | ベルントの案内で大聖堂の北翼を歩いた際、アルのデバッグスキルが無意識に反応した。足の裏から「低周波のうなりのような」感覚（微弱なノイズ）を感知した。アルがベルントに問うと「大聖堂の地下には大崩壊時代の遺物が安置されている。聖職者でも立ち入りは制限されている」という回答を得た（LOOP-009開始） |
| EVT-015-07 | 大聖堂からの帰路、フィンが大聖堂の外壁を見ながら遠い目をしていた（外面描写のみ。INV-B03準拠）。リーラがアルに「フィンのこと?」と小声で確認した |
| EVT-015-08 | 魔法学院図書館の開架棚でフィンが偶然「異才者の記録——その記述と検証」という冊子（目録外・古い革表紙）を発見した。アルが読み、三人の「異才者（Outsider）」の記録（エーリヒ＝ヴァン・ミラン・テオ＝クラッセン）を確認した。いずれも突然現れ消息を絶った記録（LOOP-001進展） |
| EVT-015-09 | アルが「彼らは何者だったのか、なぜ消えたのか」という問いを持つに至った。「自分と同じ境遇の存在が過去にもいたかもしれない」という認識が生まれた |

### 世界状態の変化

- カエルム内環が初入場・描写確定（白石の建物・幅広の通り・装飾の石刻・外環より少ない人数と異なる装い）
- 魔法学院内部が確定（白石のアーチ門・中庭の魔法陣円形石板・塔の螺旋階段・最上階の院長室・内環の全景が見える窓）
- ルクス教大聖堂の内部が確定（大礼拝堂の高窓と光の帯・北翼の廊下・執務室）
- 魔法学院図書館が確定（西棟一〜二階・開架四列・革表紙と羊皮紙綴じの混在・目録外に「異才者の記録」が存在）
- アルが内環への通行票を取得した（滞在期間中有効）
