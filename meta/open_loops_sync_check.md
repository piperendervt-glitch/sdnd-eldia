## 同期フロー

```
エピソード完成
  → QA レポート作成
  → open_loops.md 更新（ステータス・last_touched）
  → 本チェックリスト全項目 ✅ 確認
  → Canon エントリ作成 + quick_ref 更新
  → active ウィンドウ管理（archive 移動）
```

## 違反時の対応
- チェック未完了のまま Canon 確定を試みた場合、作業を中断する
- 不一致が見つかった場合、open_loops.md と quick_ref.md を修正してから再チェック
