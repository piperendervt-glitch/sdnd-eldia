# CANON-011: EP-011「Eランクの日常」正史記録（提案）

<!-- 確定日: （QA PASS後に記入） -->

## サマリー

EP-010から2日後の秋の中旬、アル・リーラ・フィンの三人がパーティとして初のEランク依頼（薬草採取）を実行した。採取地クロッカ丘陵の南斜面で土属性と水属性の遷移帯崩壊による植物の過剰マナ吸収（属性境界バグ）を発見。アルが仲間の前で初めてデバッグスキルを能動的に発動し、その能力がリーラとフィンに目撃された。帰還後ハンナがガルドへの上申を決定し、アルの能力の認知が始まった。

## 確定事実

### イベント

- EVT-011-01: EP-010から2日後の秋の中旬、アル・リーラ・フィンの三人がハンナから薬草採取の依頼（八種類、各五束以上、報酬銅貨四十枚）を受注した。パーティとして初の共同依頼。

- EVT-011-02: リーラが先導し、クロッカ丘陵南斜面の採取地に到着した。採取中、役割分担が自然に機能した（リーラ: 地形知識・群落の特定、フィン: 品質判断・魔法理論知識、アル: 摘み取りと仕分け）。リーラはクロッカ丘陵の植生を頭の中に保持しており、フィンが手帳への文書化を申し出た。

- EVT-011-03: 採取中、クロッカ丘陵南斜面の岩場の窪みから十歩ほど北の区画で、植物（カーミン草）の異常成長を発見した。茎が通常の三倍、葉が半透明で葉脈が浮き、葉縁に青白い結晶が付着していた。リーラが「空気が違う、重い・冷たい感じがする」と感じた。フィンが土属性・水属性の「属性の遷移帯」の可能性を指摘した（LOOP-003進展）。

- EVT-011-04: アルが対象（結晶化した草の群落）を視認した状態でデバッグスキルを能動的に発動した。MP消費: 基礎15 + 解析追加20 = 合計35MP。リーラとフィンがアルの「目が変わる」様子を初めて目撃した。スキル解除後、アルが一歩よろめく疲労症状あり（INV-A01準拠）。デバッグ修正は行わなかった（INV-C01・INV-D02準拠）。

- EVT-011-05: デバッグ解析の結果、アルが確認した内容:「土属性と水属性のマナ遷移帯が、正常の二十分の一以下にまで崩壊している。境界フィルタが機能不全に陥っており、植物が両属性のマナを同時に通常の七倍以上取り込んでいる（overflow状態）。処理しきれないマナが結晶として植物表面に析出している」。デバッグ出力タグ: `boundary_violation: attribute_transition_error`。

- EVT-011-06: アルが「今は修正できない」と明言した（理由: 境界フィルタ再構成はMP・理論理解が不足。副作用が不予測）。異常区画を避けて採取を完遂し、リーラが地図に場所を記録した。

- EVT-011-07: ギルドに帰還。ハンナが薬草の品質を確認し（全品合格）、報酬を支払った。リーラがアルの能力を証言し、フィンが属性境界バグの概要を説明した。ハンナが「魔法塾のそばの枯れと同じ原因かもしれない」として、ガルドへの上申を決定した。

- EVT-011-08: 夕刻、銀の麦穂亭の廊下でフィンがアルに「塾の先生にも報告する」と述べた。アルが「良いことだと思います」と答え、二人はバグ発見時の感覚について短く話した（フィン:「自分が知らない世界の深さを突然見せられた怖さ」）。アルは内心でMP残量（65/120）と次の解析計画を確認し就寝した。

### 世界状態の変化

- クロッカ丘陵南斜面に属性境界バグ（土属性・水属性の遷移帯崩壊）を確認。植物への影響（巨大化・結晶化）という新しい症状パターンが記録された。
- ハンナが当該情報をガルド（ギルド支部長）へ上申することを決定した。これはアルのデバッグスキルの能力がギルドの上層に認知される最初の機会となる。
- リンデン町周辺での魔法体系の異常が、採取地（クロッカ丘陵）にまで拡大していることが確認された。異常が魔法塾に限らず広域に分布している根拠が追加された（LOOP-003進展）。

### 新規設定

1. **クロッカ丘陵**: リンデン町の北東。街道を抜けた丘陵地帯。南斜面に薬草群落（カーミン草・ヴェール菊等）。ホーンラビットが午後から出没するため、採取は午前中が安全。ギルドから徒歩で採取地まで一定距離（詳細な時間は未確定、次話以降で確定可）。

2. **属性境界バグ（新バグ種・確定）**: RULE-M07「境界バグ」に分類される。土属性と水属性のマナの「遷移帯」が崩壊した状態。正常時は両属性の間に幅0.80m程度の緩衝帯が存在するが、崩壊時は0.04m以下まで縮小し、インピーダンス不整合が発生する（不整合率11.2倍）。植物が両属性のマナを同時に過剰吸収（通常の7倍以上）し、結晶化・巨大化という症状が発現する。デバッグ出力タグ: `boundary_violation: attribute_transition_error`。修正には境界フィルタの再構成が必要（現時点でアルの能力では不可）。

3. **薬草採取依頼の品質基準（確定）**: 傷なし・汚泥なし・摘み取りから三時間以内に持ち帰ること。夕方以降は薬草の魔力含有量が落ちる（薬効の低下）。これはギルドの標準的な品質規定として機能する。

4. **パーティとしての役割分担（確定）**: リーラ（斥候・地形知識・護衛）・フィン（魔法理論・知識担当）・アル（解析担当）という自然な役割分担が実地で初めて機能したことが確定。

5. **デバッグスキル発動の外見描写（確定）**: アルの目が「焦点を失ったわけではないが、別の何かを見ている」状態になる。琥珀色の瞳の変化として第三者（リーラ・フィン）に観察された。発動後は軽微な疲労感と一歩よろめく程度の身体症状がある（35MP消費の場合）。

## 備考

### MP状態の詳細

EP-010終了時: 90/120。
EP-011開始時: 100/120（2日間の通常睡眠で回復。極度消耗からの回復はほぼ完了）。
EP-011終了時: 65/120（デバッグスキル発動35MP消費後）。

RULE-M01補記: EP-008の極度消耗（105MP、87.5%）から約2週間が経過。90→100への回復は健常域の回復ペースと整合する。残り20MPの完全回復にはさらに1〜2日の通常睡眠が必要な状態。65/120は通常消費（総MPの70%未満の消費実績）であり、翌日には75〜80/120程度への回復が見込まれる。

### デバッグログの形式について

EP-011で使用したデバッグログ形式:
```
[SCAN: boundary_zone_anomaly]
boundary_violation: attribute_transition_error
impedance_match: FAIL (mismatch_ratio = 11.2)
absorption_overflow: TRUE (accumulated_excess: 718% of nominal capacity)
note: patch NOT applied — insufficient authority
```

EP-004のログ（NullRef・overflow）、EP-008のログ（七重フィードバックループ）とは異なる種類のバグとして、別タグ（boundary_violation）を使用。「insufficient authority」という出力はアルのスキルレベルでは修正権限がないことをシステムが示しているとも読め、LOOP-001（転生の謎）・LOOP-007（コードに見える理由）への伏線としても機能する。

### LOOP-010の進展記録

フィンは今日のデバッグスキル発動を「実物として」目撃した。これは叔父（司教）への報告に重要な情報を追加することになる。ただし、フィンがアルとの会話で「塾の先生にも報告する」とのみ言及したのは、アルへの誠実さと叔父への義務の間の板挟みを示す。EP-012では、このデバッグスキル目撃情報を叔父に報告すべきか否かでフィンが葛藤することが予定されている。
